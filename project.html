      Инструкция по странице проекта

      • Основное назначение:

        Эта страница предназначена для управления проектом и поиска оптимального времени для общих встреч или тренировок.

      • Календарь и цветовая индикация:

        Календарь анализирует занятость участников и подсвечивает дни, подходящие для встречи:

        Идеальный день - Есть 2-часовой промежуток, когда все участники свободны.

        Хороший день - Есть 2-часовой промежуток, когда почти все свободны, а один участник не определился со статусом.

        Возможный день - Есть 2-часовой промежуток, когда почти все свободны, а один участник занят.

      • Сводная таблица занятости:

        При выборе дня в календаре, в этой таблице отображается детальная занятость каждого участника по часам. Цвета означают:

        Статус участника (зеленый - свободен, красный - занят, желтый - не уверен).

        Тренировка - На этот час у участника назначена тренировка в каком-либо проекте.

      • Управление проектом:

        Пользователи с ролью Администратор или Ответственный за проект имеют расширенные права на управление.

        • Тренировки: Добавляйте, редактируйте и удаляйте тренировки для этого проекта. Указывайте время, место и комментарии.
        • Материалы: Этот виджет (можно свернуть/развернуть) предназначен для ведения учета по партиям, костюмам и задачам для каждого участника. Управлять материалами могут администратор и ответственный.
          • Партия: Добавление музыкальной или сценической партии с описанием и фотографиями.
          • Костюм: Учет элементов костюма с названием, ссылкой на товар и фотографиями.
          • Задача: Постановка индивидуальных задач с описанием и возможностью отмечать их выполнение.
        • Управление для администратора:
          • Участники: Добавляйте или удаляйте участников из проекта (виджет можно свернуть/развернуть). При удалении участника все связанные с ним материалы также будут удалены.
          • Ответственный: Назначьте одного из участников ответственным за проект. Ответственный получает права на управление тренировками и материалами.
