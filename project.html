      Инструкция по странице проекта

      • Основное назначение:

        Эта страница предназначена для управления проектом и поиска оптимального времени для общих встреч или тренировок.

      • Календарь и цветовая индикация:

        Календарь анализирует занятость участников и подсвечивает дни, подходящие для встречи:

        Идеальный день - Есть 2-часовой промежуток, когда все участники свободны.

        Хороший день - Есть 2-часовой промежуток, когда почти все свободны, а один участник не определился со статусом.

        Возможный день - Есть 2-часовой промежуток, когда почти все свободны, а один участник занят.

      • Сводная таблица занятости:

        При выборе дня в календаре, в этой таблице отображается детальная занятость каждого участника по часам. Цвета означают:

        Статус участника (зеленый - свободен, красный - занят, желтый - не уверен).

        Тренировка - На этот час у участника назначена тренировка в каком-либо проекте.

        Конфликт (Занят) - Тренировка пересекается с часом, когда участник отметил, что он занят.

        Конфликт (Не уверен) - Тренировка пересекается с часом, где участник не определился со статусом.

      • Управление проектом (для администратора и ответственного):
        • Тренировки: Добавляйте, редактируйте и удаляйте тренировки для этого проекта.
        • Партии: Назначайте роли или музыкальные партии для каждого участника (виджет можно свернуть/развернуть).
        • Участники: Добавляйте или удаляйте участников из проекта (виджет можно свернуть/развернуть).
        • Ответственный: Назначьте одного из участников ответственным за проект. Ответственный сможет видеть эту страницу и управлять тренировками.
